PROBLEM:
A 2.0 L container holds an ideal gas at 27°C and 1.5 atm pressure. The gas is heated to 127°C while the volume is allowed to expand to 3.0 L. Calculate the final pressure of the gas.

THINKING:
  Step 1: Identify the known variables: initial volume V1 = 2.0 L, initial pressure P1 = 1.5 atm, initial temperature T1 = 27°C (convert to Kelvin), final volume V2 = 3.0 L, final temperature T2 = 127°C (convert to Kelvin).
  Step 2: Convert temperatures from Celsius to Kelvin by adding 273.15: T1 = 27 + 273.15 = 300.15 K, T2 = 127 + 273.15 = 400.15 K.
  Step 3: Since the gas is ideal and the amount of gas remains constant, apply the combined gas law: (P1 * V1) / T1 = (P2 * V2) / T2.
  Step 4: Rearrange the combined gas law to solve for the final pressure P2: P2 = (P1 * V1 * T2) / (V2 * T1).
  Step 5: Substitute known values into the equation: P2 = (1.5 atm * 2.0 L * 400.15 K) / (3.0 L * 300.15 K).
  Step 6: Calculate P2 numerically and determine the final pressure.

SOLUTION:
P2 = (1.5 * 2.0 * 400.15) / (3.0 * 300.15) = (1.5 * 2.0 * 400.15) / 900.45 ≈ (1.5 * 800.3) / 900.45 = 1200.45 / 900.45 ≈ 1.33 atm. Thus, the final pressure of the gas after heating and expansion is approximately 1.33 atm.

VERIFICATION:
Check units and logic: The volume increased while temperature increased, so pressure might increase or decrease depending on relative changes. Here, volume increased by 50% (2.0 to 3.0 L) but temperature increased by ~33%. The pressure decreased from 1.5 atm to 1.33 atm, which makes sense because volume increased more proportionally than temperature. The calculations are consistent and units cancel correctly.

PROBLEM:
A farmer has a rectangular field that is 120 meters long and 80 meters wide. He wants to increase both the length and the width by the same amount so that the area of the new field is exactly double the original area. By how many meters should he increase each dimension?

THINKING:
  Step 1: Calculate the original area of the field by multiplying length and width: 120 m × 80 m.
  Step 2: Determine the target area, which is double the original area.
  Step 3: Let x be the amount (in meters) the farmer increases both length and width.
  Step 4: Write an equation for the new area: (120 + x)(80 + x) = 2 × original area.
  Step 5: Expand and simplify the equation to form a quadratic in terms of x.
  Step 6: Solve the quadratic equation for x, and choose the positive root since length increase must be positive.

SOLUTION:
The original area is 120 × 80 = 9,600 m². The target area is 2 × 9,600 = 19,200 m². Set up the equation: (120 + x)(80 + x) = 19,200. Expanding: 120×80 + 120x + 80x + x² = 19,200 → 9,600 + 200x + x² = 19,200. Rearranged: x² + 200x + 9,600 - 19,200 = 0 → x² + 200x - 9,600 = 0. Using the quadratic formula x = [-200 ± √(200² - 4×1×(-9,600))]/2. Calculate discriminant: 40,000 + 38,400 = 78,400. √78,400 = 280. So, x = [-200 ± 280]/2. Possible roots: (80)/2 = 40 or (-480)/2 = -240. Negative length increase is not valid, so x = 40 meters. The farmer should increase each dimension by 40 meters.

VERIFICATION:
Check new dimensions: 120 + 40 = 160 m, 80 + 40 = 120 m. New area: 160 × 120 = 19,200 m², which is exactly double the original 9,600 m². Therefore, the answer is correct.

The empirical formula is MgO because the mole ratio of magnesium to oxygen is approximately 1:1, indicating one atom of magnesium combines with one atom of oxygen to form magnesium oxide.

VERIFICATION:
Re-checking mole calculations and ratios confirms that the mass to mole conversions and ratio simplifications are correct, supporting the empirical formula MgO.

PROBLEM:
A farmer has a rectangular field that is twice as long as it is wide. He wants to increase the area of the field by 50% by increasing both the length and width by the same number of meters. How many meters should he increase each dimension to achieve this?

THINKING:
  Step 1: Assign variables. Let the width be x meters, so the length is 2x meters.
  Step 2: Express the original area: Area_original = length × width = 2x × x = 2x².
  Step 3: Let the increase in each dimension be y meters. Then new dimensions are (x + y) and (2x + y). The new area is (x + y)(2x + y).
  Step 4: Set up the equation for the increased area: (x + y)(2x + y) = 1.5 × 2x² = 3x², then solve for y in terms of x.

SOLUTION:
Expanding (x + y)(2x + y) gives 2x² + x y + 2x y + y² = 2x² + 3x y + y². Set this equal to 3x²: 2x² + 3x y + y² = 3x². Rearranged: 3x y + y² = 3x² - 2x² = x². Divide both sides by x: 3 y + (y²)/x = x. Substitute t = y/x for simplicity: 3 t + t² = 1. Rearranged: t² + 3 t - 1 = 0. Solve quadratic: t = [-3 ± sqrt(9 + 4)] / 2 = [-3 ± sqrt(13)] / 2. Since y must be positive, take positive root: t = (-3 + 3.606) / 2 ≈ 0.303. Therefore y = t x ≈ 0.303 x. So, the farmer should increase each dimension by about 30.3% of the original width to increase the area by 50%.

VERIFICATION:
If x = 10 m, then y ≈ 0.303 × 10 = 3.03 m. Original area = 2 × 10² = 200 m². New dimensions: width = 13.03 m, length = 23.03 m. New area ≈ 13.03 × 23.03 = 300 m², which is exactly 1.5 times the original area, confirming the solution.

PROBLEM:
A 5-gram piece of metal at 100°C is dropped into 50 grams of water at 20°C. If the final temperature of the system is 22°C, what is the specific heat capacity of the metal? (Specific heat capacity of water = 4.18 J/g°C)

THINKING:
  Step 1: Calculate the heat gained by the water using Q = mcΔT, where m = 50 g, c = 4.18 J/g°C, and ΔT = 22°C - 20°C = 2°C.
  Step 2: Assume the heat lost by the metal equals the heat gained by the water. Use Q = mcΔT for the metal, where m = 5 g, ΔT = 100°C - 22°C = 78°C, and solve for c (specific heat capacity of the metal).

SOLUTION:
Heat gained by water: Q = 50 g × 4.18 J/g°C × 2°C = 418 J. Heat lost by metal = 418 J. Using Q = mcΔT for metal: 418 J = 5 g × c × 78°C, so c = 418 J / (5 g × 78°C) ≈ 1.07 J/g°C. Thus, the metal's specific heat capacity is approximately 1.07 J/g°C.

VERIFICATION:
Check units and calculations: 5 × 1.07 × 78 ≈ 418 J, which matches the heat gained by water. Units are consistent (Joules), confirming the solution is correct.

  Step 1: Calculate the original area of the field by multiplying length and width: 150 m × 80 m.
  Step 2: Determine the new area by increasing the original area by 25%. This means multiplying the original area by 1.25.
  Step 3: Let x be the number of meters added to both length and width. Then the new dimensions are (150 + x) and (80 + x). Set up the equation (150 + x)(80 + x) = 1.25 × original area.

Calculate discriminant: 230² = 52,900; 4×3,000=12,000; sum=64,900. sqrt(64,900) ≈ 254.8. Then x = [-230 ± 254.8]/2. Positive root: (-230 + 254.8)/2 = 24.8/2 = 12.4. So, the farmer should add approximately 12.4 meters to both length and width.

Given a list of integers, write a Python function that returns the first duplicate number it encounters. If no duplicates are found, return -1.

THINKING:
  Step 1: Iterate through the list while keeping track of numbers seen so far using a set.
  Step 2: For each number, check if it is already in the set. If yes, return that number immediately as the first duplicate; if not, add it to the set and continue.

SOLUTION:
def first_duplicate(nums):
    seen = set()
    for num in nums:
        if num in seen:
            return num
        seen.add(num)
    return -1

Explanation: The set 'seen' allows O(1) membership checks. As soon as a number repeats, it is returned, ensuring the first duplicate is found efficiently.

def factorial(n):

    if n == 0:
        return 1
